Страница 2, полезные инструменты для дизайнеров

  • TinyPNG (Бесплатно/Платно) TinyPNG — инструмент для сжатия изображений без потери качества. Это очень полезно для веб-дизайнеров, чтобы уменьшить размер изображений на сайте и ускорить загрузку.
  • Balsamiq (Платно) Balsamiq — инструмент для быстрого прототипирования интерфейсов. С помощью Balsamiq ты можешь создавать черновые макеты и тестировать идеи, не тратя много времени на детали.
  • Trello (Бесплатно/Платно) Trello — онлайн-платформа для управления проектами. Очень полезен для дизайнеров, работающих в команде, для организации задач, обмена идеями и отслеживания прогресса.
  • Gravit Designer (Бесплатно/Платно) Gravit Designer — бесплатный и удобный векторный графический редактор, который можно использовать для создания логотипов, иконок и других графических элементов.
  • Sketch (Платно) Sketch — популярный инструмент для веб-дизайнеров и дизайнеров интерфейсов. Это мощный редактор, который часто используется для создания интерфейсов и прототипов, но работает только на Mac.
  • InVision (Бесплатно/Платно) InVision — инструмент для создания прототипов и дизайнов интерфейсов. В нем можно делать анимации, интегрировать с другими сервисами и собирать отзывы от пользователей.
  • Looka (Бесплатно/Платно) Looka — онлайн-сервис для создания логотипов. Можно легко и быстро сгенерировать логотип для проекта, используя шаблоны и настроив их под свои нужды.
  • Placeit (Платно) Placeit — сервис для создания мокапов (визуальных моделей). Это позволяет быстро показать, как твой дизайн будет выглядеть на реальных продуктах, например, футболках, визитках или экранах смартфонов.
  • Iconmonstr (Бесплатно) Iconmonstr — библиотека бесплатных иконок, которые можно использовать в проектах. Все иконки доступны в SVG-формате, что делает их удобными для веб-дизайна.
  • Webflow (Бесплатно/Платно) Webflow — мощный инструмент для создания веб-сайтов с визуальным редактором. Он позволяет дизайнерам без навыков кодирования создавать адаптивные сайты с помощью перетаскивания элементов.
  • Zeplin (Платно) Zeplin — инструмент для совместной работы между дизайнерами и разработчиками. Он упрощает передачу дизайна от дизайнера к разработчику, генерируя код CSS и делая макеты доступными для команд.
  • Photopea (Бесплатно) Photopea — бесплатный онлайн-редактор изображений, который поддерживает формат PSD (файлы Photoshop). Отлично подходит для редактирования изображений и создания графических элементов.
